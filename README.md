# 🩺 HealthMate - Personalized Health & Wellness Platform

HealthMate is a full-stack health and wellness web application that provides personalized diet recommendations, wellness guidance, and expert-backed nutrition support based on users' health goals, lifestyle, and dietary preferences.

## 🚀 Features

### 👤 User Module
- User Registration & Login
- Health Assessment Quiz
- Personalized Diet Recommendations
- Save & View Diet Plans
- Wellness Guidance
- Ratings & Feedback

### 🥗 Nutritionist Module
- Nutritionist Registration & Login
- Create & Manage Diet Plans
- Expert Wellness Recommendations
- Credential Verification

### 🛠️ Admin Module
- Admin Login
- User Management
- Nutritionist Verification
- Feedback Monitoring

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Apache (XAMPP)

## 📂 How to Run Locally

1. Install XAMPP
2. Place the project folder inside `htdocs`
3. Import the database using phpMyAdmin
4. Start Apache and MySQL
5. Open:

```text
http://localhost/HealthMate
```
